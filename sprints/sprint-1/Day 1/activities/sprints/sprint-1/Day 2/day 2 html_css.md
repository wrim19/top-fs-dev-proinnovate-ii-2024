# Bootcamp Day 2

## Overview
Welcome to Day 2 of the bootcamp! Today, we’ll dive into front-end development fundamentals. You’ll learn about the importance of writing semantic HTML, implementing responsive web design, and mastering CSS layouts using Flexbox and Grid.

## Schedule

### Morning: Semantic Web and Best Practices
In the morning session, you’ll focus on the importance of semantic HTML and writing clean, maintainable code. Following that, we’ll introduce best practices for building responsive websites.

- **8:00 AM - 9:00 AM**: **Introduction to Semantic HTML**
  - What is Semantic HTML and why it matters
  - Key semantic elements (`<header>`, `<nav>`, `<article>`, `<section>`, `<footer>`)
  - Best practices for writing accessible and SEO-friendly HTML

- **9:00 AM - 10:00 AM**: **Best Practices in Web Development**
  - Keeping your HTML and CSS clean and organized
  - Importance of using external stylesheets and avoiding inline styles
  - Best practices for naming conventions (BEM methodology)
  - Writing reusable and scalable CSS

- **10:00 AM - 11:00 AM**: **Introduction to Responsive Design**
  - What is responsive design and why it’s crucial
  - Key concepts: viewport, media queries, breakpoints
  - Mobile-first design approach

- **11:00 AM - 12:00 PM**: **Implementing Responsive Design**
  - How to use media queries effectively
  - Best practices for handling different screen sizes and devices
  - Working with responsive typography and images

### Afternoon: CSS Layouts, Flexbox, and Grid
In the afternoon, you’ll get hands-on experience with CSS layouts, Flexbox, and Grid. You’ll learn how to create modern layouts and practice designing responsive grids.

- **1:00 PM - 2:00 PM**: **Introduction to CSS Layouts**
  - Overview of different layout techniques (floats, inline-blocks, Flexbox, Grid)
  - Advantages of Flexbox and Grid over older layout methods
  - When to use Flexbox vs. Grid

- **2:00 PM - 3:00 PM**: **Deep Dive into Flexbox**
  - Core concepts: flex container, flex items, flex direction, and alignment
  - Building layouts with Flexbox (examples and hands-on exercises)

- **3:00 PM - 4:00 PM**: **Mastering CSS Grid**
  - Core concepts: grid container, grid items, grid tracks (rows and columns)
  - Defining grid areas, placing items, and creating complex layouts
  - Practical examples and exercises with CSS Grid

- **4:00 PM - 5:00 PM**: **Practice and Build**
  - Build a responsive layout using both Flexbox and Grid
  - Combine what you've learned to create a modern, responsive web page
  - Review and share your work with your peers for feedback

## Activities

### Semantic HTML Practice
- Refactor an existing HTML page to use semantic elements
- Create a simple webpage structure using semantic elements (e.g., header, navigation, main content, and footer)

### Best Practices
- Organize your CSS using the BEM naming convention
- Refactor inline styles into external stylesheets

### Responsive Design
- Add responsive media queries to an existing website
- Practice creating fluid layouts with percentage-based widths
- Implement responsive images using `srcset`

### Flexbox and Grid Practice
- Build a navigation bar using Flexbox
- Create a product listing layout using Flexbox for alignment and spacing
- Design a responsive grid-based layout for a photo gallery or a blog
